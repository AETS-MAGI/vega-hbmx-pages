1. NumPy は何か

1. What NumPy Is

ひとことで言うと、Python で高速に多次元配列を扱うための基盤ライブラリです。

In one sentence: the foundational Python library for fast multi-dimensional array computing.

NumPy は、リストより効率よく数値データを扱うための ndarray を提供します。線形代数、統計、前処理などの多くが NumPy で表現できます。

NumPy provides ndarray for efficient numeric data handling beyond Python lists. Many tasks in linear algebra, statistics, and preprocessing can be expressed with NumPy.

覚える最小セット: ndarray / shape / dtype / axis / broadcasting

Minimal set to remember: ndarray / shape / dtype / axis / broadcasting

3. 最短サンプル（10行）

3. Shortest Sample (10 lines)

import numpy as np a = np.array([[1, 2, 3], [4, 5, 6]], dtype=np.float32) b = np.array([[7, 8], [9, 10], [11, 12]], dtype=np.float32) c = a @ b print(a.shape, b.shape, c.shape) print(c)

この 10 行で「配列を作る」「shape を確認する」「行列積を計算する」ができます。PyTorch のテンソル計算と読み方がかなり近いので、先にここを通ると学習がスムーズです。

These 10 lines already cover array creation, shape inspection, and matrix multiplication. The reading style is close to PyTorch tensor math, making later learning smoother.

4. 初学者が詰まりやすい点

4. Common Beginner Stumbles

よくある詰まり: 1) shape を見ないまま計算する 2) axis の意味を取り違える 3) list と ndarray を混同する

Common stumbles: 1) Computing without checking shape 2) Misreading axis semantics 3) Mixing list and ndarray mental models

5. NumPyとPyTorchの関係

5. NumPy and PyTorch Relationship

NumPy は CPU 側の配列計算基盤、PyTorch はそれを拡張して自動微分や GPU 実行を持つ学習向け基盤、と考えると把握しやすいです。つまり、NumPy は PyTorch を読むための語彙づくりに向いています。

A useful mental model: NumPy is the CPU-side array foundation, while PyTorch extends this style with autograd and GPU execution for learning workflows. NumPy builds vocabulary for reading PyTorch code.
